What we will be covering in this workshop

  1. Downloading Bootstrap and incorporating it into an HTML document
  2. Working with the Bootstrap responsive grid system
  3. Exploring Bootstrap’s core CSS, overriding this CSS using standard CSS techniques
  4. Incorporating Bootstrap’s image formatting elements and icons
  5. Creating and styling fully responsive interactive menus with Bootstrap.

What this workshopt will NOT be covering

  1. Creating fully customized 'themes' for Bootstrap
  2. Designing an app

What is Bootstrap?

Twitter's Bootstrap is a frontend UI (user interface) framework that speeds up the development of new websites and web applications.

It is the #1 project on github with 65,000+ stars and over 23,000 forks

Bootstrap is an open source project on GitHub, meaning anyone can contribute to it’s progress.

It has been used by NASA & MSNBC
